The Primal is Elements  project provide three modules, each module is  python package
   - **PrimalCore python** package :
      - Implementation of catalogs and ML datasets
      - ML models based on regressors and classifiers
      - Validation and Model optimization
      - PDF for PhZ

   - **PrimalInteractive** python package:
      - tools for interactive analysis

   - **PrimalPipeline** python package:
      - implemented pipelines for ML  PhZ tasks
